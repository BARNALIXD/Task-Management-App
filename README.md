Task Management Application
A full-stack app for managing tasks by ROI with React (frontend), Express.js (backend), Prisma ORM, and NeonDB/PostgreSQL.

Live Demo: https://task-management-appxd1128.netlify.app/

Features
Add, edit, delete, and view tasks

Filter, search, and sort by status, priority, and ROI

Undo delete and CSV import/export

Summary insights: total revenue, efficiency, performance grade

Persistent PostgreSQL storage

Tech Stack
Frontend: React 18

Backend: Express.js

ORM: Prisma

Database: NeonDB (PostgreSQL)
